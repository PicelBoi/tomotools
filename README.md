# TomoTools

A set of open-source tools to help with Mii creation for the Tomodachi Life series of games. Made with Astro.js.

AI was not used in the development of this project.

## Tools used

- Astro.js
- Desmos
- TI-84 Plus
- Nintendo Switch
- Astro.js
- iPhone 13 mini
- [Online Mic Test's Pitch Detector](https://www.onlinemictest.com/tuners/pitch-detector/)
- [pitchy](https://www.npmjs.com/package/pitchy)
